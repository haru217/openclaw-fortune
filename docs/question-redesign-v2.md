# 質問設計 v2（2026-03-24）

## 設計方針
- Q1ラベル: 「相談したいテーマは？」（全モジュール共通）
- Q2ラベル: Q1ごとに2種類から選択
  - 「知りたいことは？」— 質問系のQ2
  - 「もう少し教えてください」— 状況・感情系のQ2
- Q3: 自由記述（Q1ごとにhintを変更）
- LINE cardの制約: 選択肢13文字以内、4択固定
- 占いが効く「答えのない問い」を優先
- サブカテゴリ名変更: love:start「恋の始まり」→「片思い・出会い」

## Q2ラベル割り当て

### 「知りたいことは？」を使うQ1:
- work:career — 今の仕事が合わない気がする / 独立・副業(これから→Q3) / 独立・副業(すでに→Q3) / 働き始めたい・復帰したい / キャリアプランについて
- work:people — この人との相性 / いつ状況が変わるか / 我慢か離脱か
- love:start — 気になる人と進展しない / 好きだけど踏み出せない / 出会いがない / 婚活中だが決め手がない
- love:partner — 相手の気持ち / 二人の未来 / 別れるべきか続けるべきか / 二人の相性
- love:reunion — 復縁の可能性 / 動くべきタイミング
- relation:family — 家族の本音が知りたい / 干渉や束縛がつらい

### 「もう少し教えてください」を使うQ1:
- work:people — 相手の本音を知りたい
- love:reunion — 相手の今の気持ち / この恋を諦めるべきか
- relation:family — 離れたいけど罪悪感がある / 家族の中で居場所がない
- relation:friend — 友達の本音が知りたい / 距離を取るべきか迷う / グループで居心地が悪い / 気を遣いすぎて疲れる

### 特殊:
- work:career「独立・副業について」→ Q2ラベルは「状況は？」（これから始めたい / すでに独立・副業中の2択分岐）
